# KO Simple Dashboard

A very simple static dashboard for the KO capstone project.

## Easiest GitHub setup
Upload these files directly to your repository root and enable GitHub Pages.

Files you should see in the repo root:
- `index.html`
- `styles.css`
- `app.js`
- `dashboard_data.json`

## GitHub Pages
1. Upload the files to your repo.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose **main** and **/root**.
5. Save.
6. Wait 1–2 minutes for your `github.io` URL.
